日初学堂是志愿者之间的分享会

日初学堂

这个项目最初的起意是志愿者之间的分享会，日初的志愿者来自五湖四海，有医院的医生药剂师、外企的人力资源主管财务总监、高校教师、还有来自媒体、化工、保险、体育用品、教育、外贸、新能源汽车研发、智慧办公等各行各业的人士，拥有不同的文化和专业背景。

除此以外，大家还有丰富的业余爱好，有人热衷于研究植物，有人爱好声乐，有人喜欢心理分析，有人熟悉精油/中医/彩妆，有人酷爱瑜伽，有人热爱手工，也有人爱好烹饪，日初学堂作为志愿者之间的一个特别活动，既可以加深志愿者之间的互相了解，也可以让大家在短时间内有一些不同于平常的体验，丰富知识的同时也陶冶情操。

多次实践之后，日初学堂视 内容而向其他捐助人及爱心人士开放，志愿者们以公益讲座或工作坊的形式将自己喜欢或擅 长领域的知识分享给更多的人。2018年春天举办的“正面管教家长课”活动及2019年年底举办 的“身体和心灵，两个都要在路上-成长型思维与颈椎保护”活动，都曾获得广泛好评。每次 活动筹备结束后都会发布在日初公众号，届时欢迎通过网络报名的形式进行参与。
